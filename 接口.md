## 接口

###  完全遵循restful风格

- 请求成功，返回 http code 200，数据从response 获取

- 请求失败，返回http code 4xx,  前端 alert 一下

  前端无需了解code含义，举例：ajax，成功走success，失败走error， 获取response data 即可

  

### 预约

POST    /api/order/create

参数：

- `room_id` 必选 , 数组,  通过（ /api/rooms?language=zh）接口获取房间信息 对应界面选择选择房间那些项
- `number` 必填，用餐人数
- `budget` 必填，人均预算
- `dinner_time` 必填，用餐时间， 时间格式 '%Y-%m-%d %H:%M:S'
- `phone` 必填 手机号
- `email`  必填 邮箱
- `taboo_food_id` 可选，忌口食物 ，选择配菜，通过（ /api/foods?language=zh&type=chives_garnish&type=vegetarian_garnish）接口获取配菜列表 对应界面选择忌口的那些项
- `avoid_food` 可选，对应界面特殊喜爱和忌口
- `spicy` 可选。接受的辣度 。取值为 low|normal|high

```json
{
	"room_id":[1],
	"number":4,  
	"budget": 100, 
	"dinner_time": "2019-07-17 15:15:15", 
	"phone": 13585853274, 
	"email": "a@qq.com",
    "taboo_food_id": [1],
    "avoid_food": "123",
    "spicy": "low"
}
```

响应:

```json
{
    "_status": "OK",
    "data": {
        "is_completed": null,
        "room_id": 1,
        "number": 4,
        "budget": 100,
        "dinner_time": "2019-07-17 15:15:15",
        "phone": 13585853274,
        "email": "a@qq.com",
        "id": 1
    }
}
```




**语言选择：** 

```
    zh = '中文'
    en = '英文'
    gm = '德语'
```

### 菜品信息

GET   /api/foods

**参数**：

- `language`   必选 , 取值 zh|en|gm
- `type` 可选 ，取值chives|vegetarian|chives_garnish|vegetarian_garnish, **注：就这个type支持多选**

**举例**：  /api/foods?language=zh&type=chives&type=vegetarian

注：type类型：

```
chives = '荤菜'
vegetarian = '素菜'

chives_garnish = '荤类配菜'
vegetarian_garnish = '素类配菜'
```
